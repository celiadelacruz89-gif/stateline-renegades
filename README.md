# Stateline Renegades (Clean Starter)

This is a clean Vite + React starter site designed to deploy easily on Vercel.

## Deploy on Vercel (recommended)
1. Push these files to your GitHub repo.
2. In Vercel: Add New Project → Import the repo.
3. Framework Preset: **Vite**
4. Build Command: **npm run build**
5. Output Directory: **dist**
6. Install Command: **npm install**
7. Deploy

## Google Search Console verification
Edit `index.html` and replace:
`YOUR_VERIFICATION_CODE_HERE` with your real code.

Then redeploy, and click Verify in Search Console.

## Local dev
```bash
npm install
npm run dev
```
